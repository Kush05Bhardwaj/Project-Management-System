# Mini Project Management System (Mini PMS)

A role-based Project Management System built with Flask and MongoDB.

## Features
- JWT Authentication
- Role-based access (Student, Mentor, Admin)
- Team management
- Document upload & review
- Evaluation system
- Secure file handling
- Dockerized backend

## Tech Stack
- Flask
- MongoDB
- JWT
- Docker

## Setup
```bash
pip install -r requirements.txt
python main.py
